Challenges

Maze Escape Creator

In this project, you are able to use the elements at the bottom of the screen to create a maze for you or another player to create and play through. There are many different obstacles and gadgets you can choose to use like a circle barrier, spinner death bar, and a door that is opened with a button that needs to be pressed down.

View Project

Challenges

Some challenges that I learned here include trying to create a function to open a certain door with a certain button, as well as trying to figure out the specific requirements in order to create a working collision function.

Lessons Learned

Here I learned that in order to create a working collision function you need to give the character some time to function through slack when it collides with an object, in order to give the program enough time to function correctly. I also learned that you need to create the doors and buttons as objects related to each other through variables and lists, which has become a tactic I have used in some other of my creative tasks.

Minesweeper Replica

This program is a replica of a known computer game called "Minesweeper", where you try to avoid hitting any bombs while clearing out the safe spaces of the "battlefield". In this program, you can play minesweeper with many different numbers of boxes, in order to regulate the difficulty of the game. You can also mark which spots you believe are bombs, so that you can avoid them at all costs.

View Project

Challenges

One challenge I faced in programming this game was first figuring out how to create the grid of blocks! After figuring that out, another challenge I faced was in how I would generate the boxes to be either bomb spaces, safe spaces, or numbered spaces, which was done by using a random generator and a detection system to track every block's type.

Lessons Learned

Here, I learned lessons that helped me create the final creative task in this portfolio through understanding how to make a detection system and a grid of blocks to work with.
